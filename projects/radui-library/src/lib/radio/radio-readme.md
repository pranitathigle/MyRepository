# Radio

<p>Syntax to use this is</p>

+ Default Radio
<code>
    `<-radio orientation="vertical" radioValues="Male, Female">
    </-radio>`
</code>
+ Horizontal Radio
<code>
    `<-radio orientation="horizontal">
    </-radio>`
</code>
+ Vertical Radio
<code>
    `<-radio orientation="vertical">
    </-radio>`
</code>
